# 👤 Face Recognition Attendance System

An intelligent attendance system that uses **face recognition technology** to automatically mark attendance in real time using AI and Machine Learning .

This project eliminates manual attendance processes and improves accuracy, efficiency, and security.

The main purpose of this project is to mark attendance automatically using face recognition instead of manual entry.

---

## 🚀 Project Overview

This system captures facial images through a camera, compares it with the stored dataset, detects and recognizes faces, and if the face matches,marks attendance automatically in a database or file system.

---

## 🧩 Features

- 📷 Real-time face detection using webcam
- 🧠 Face recognition using trained models
- 📝 Automatic attendance marking
- 📅 Date & time tracking
- ⚡ Fast and contactless system

---

## 🛠️ Tech Stack

- AI

---

## ⚙️ How It Works

1. Capture face images of users
2. Train the model using stored images
3. Start webcam for live detection
4. Recognize faces in real-time
5. Mark attendance with timestamp
